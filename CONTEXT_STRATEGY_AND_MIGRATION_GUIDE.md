# СТРАТЕГИЯ РАБОТЫ С КОНТЕКСТОМ И ПЛАН ПРОДОЛЖЕНИЯ

## ЧАСТЬ 1: ЧТО ТАКОЕ КОНТЕКСТ И ТОКЕНЫ - ПОДРОБНО

### Базовое понимание

**Контекстное окно (Context Window):**
- Это "оперативная память" Claude в рамках ОДНОГО чата
- В Claude Sonnet 4.5 контекстное окно = **200,000 токенов**
- Это включает ВСЁ: 
  - Системные инструкции (~45,000 токенов)
  - Ваши сообщения
  - Мои ответы
  - Прикрепленные файлы
  - История диалога

**Что такое токены - детально:**

Токен - это базовая единица текста для AI моделей.

**Примеры токенизации:**

Английский:
- "Hello world" = 2 токена ["Hello", " world"]
- "Hello, world!" = 4 токена ["Hello", ",", " world", "!"]
- 1 слово = обычно 1-2 токена

Русский (кириллица сложнее):
- "Привет мир" = 3-4 токена
- "Привет, мир!" = 5-6 токенов  
- 1 слово = обычно 1.5-2 токена

Код:
- `def hello():` = 5 токенов
- `    return "Hello"` = 5 токенов
- 1 строка кода = обычно 5-15 токенов

**Практические расчёты:**
```
1,000 токенов ≈ 750 английских слов ≈ 500 русских слов
10,000 токенов ≈ 7,500 английских слов ≈ 5,000 русских слов
68,000 токенов ≈ 51,000 английских слов ≈ 34,000-40,000 русских слов

Это примерно:
- 50-70 страниц A4 текста
- 20-25 детальных skill описаний
- 100-150 страниц кода
```

### Текущая ситуация В ЭТОМ чате

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
КОНТЕКСТ ЧАТА 1 (текущий)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Бюджет токенов: 190,000
Использовано: ~133,000 токенов
Осталось: ~57,000 токенов

ЧТО ИСПОЛЬЗОВАНО:
├─ Системный промпт Claude: ~45,000 токенов
├─ Ваши сообщения: ~5,000 токенов
├─ Мои ответы: ~50,000 токенов
└─ Созданные файлы в контексте: ~33,000 токенов

ЧТО ОСТАЛОСЬ:
57,000 токенов = ~40,000 русских слов
              = ~60 страниц A4
              = ~5-6 детальных skill описаний
              = ~20 кратких описаний
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Почему это критически важно понимать

**Что происходит когда контекст заканчивается:**

1. **В mid-conversation (во время ответа):**
   ```
   Claude: "Сейчас опишу Skill #25..."
   [КОНТЕКСТ ЗАПОЛНЕН]
   Claude: [Не может продолжить]
   ```
   Ответ обрывается. Нужно начинать новый чат.

2. **При попытке отправить новое сообщение:**
   ```
   Вы: "Продолжай со Skill #26"
   Система: "Context limit exceeded. Please start a new conversation."
   ```

3. **Что теряется:**
   ❌ Вся история диалога (Claude не помнит что было)
   ❌ Все созданные файлы (если не скачаны)
   ❌ Контекст проекта
   ❌ Стиль и договоренности

**ВАЖНО:** Claude НЕ ИМЕЕТ памяти между разными чатами!
Каждый новый чат = чистый лист для Claude.

## ЧАСТЬ 2: РАСЧЕТ ВОЗМОЖНОСТЕЙ ДЛЯ ТЕКУЩЕГО ЧАТА

### Детальный расчёт

**Доступно: 57,000 токенов**

**Сколько это в skills:**

1 детальный skill (~7,000 русских слов + код) = ~11,000 токенов
57,000 / 11,000 = **5.2 skills детально**

1 краткий skill (~500 слов) = ~800 токенов  
57,000 / 800 = **71 skill кратко**

## ВОПРОСЫ И ОТВЕТЫ

**Q: Что если контекст закончится посреди skill?**
A: См. "Проблема 1" в Части 6. Продолжите в новом чате с прикреплением частичного описания.

**Q: Как понять сколько токенов осталось?**
A: Claude показывает в system warnings. Также ~7,000 слов = ~11,000 токенов.

**Q: Можно ли менять структуру описаний между чатами?**
A: Можно, но нужно обновить STYLE_GUIDE и пометить в CHANGELOG.

**Q: Что делать если забыл скачать файлы из старого чата?**
A: Старые чаты доступны в истории claude.ai. Зайдите в чат и скачайте файлы.

**Q: Нужно ли описывать все 87 skills детально?**
A: Не обязательно. Можно топ-30 детально, остальные кратко. Зависит от целей.

**Q: Сколько времени займёт весь проект?**
A: При темпе 5 skills/день = ~18 дней. При 2-3 skills/день = ~30-40 дней.
