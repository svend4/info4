# 📦 Chat Migration Assistant - Обе Версии

## 🎯 КАКУЮ ВЕРСИЮ ВЫБРАТЬ?

### ✅ Используйте v1.0 БАЗОВУЮ если:
- Проект простой (1-3 чата, до 20 items)
- Работаете в одиночку
- Нужен быстрый старт без изучения
- Не требуется автоматизация
- Впервые используете checkpoint систему

### ✅ Используйте v2.0 PRO если:
- Проект большой (5+ чатов, 50+ items)
- Работаете в команде
- Нужна автоматизация (экономия 75% времени)
- Требуется Git версионирование
- Нужны метрики и аналитика
- Интеграция с Notion/Trello/Jira
- Enterprise требования

## 📖 КАК УСТАНОВИТЬ

### ШАГ 1: Выберите версию

**Для начинающих:** Установите v1.0 БАЗОВУЮ  
**Для профи:** Установите v2.0 PRO  
**Можно обе:** Установите обе версии (они не конфликтуют!)

### ШАГ 2: Установка в Claude

1. Откройте https://claude.ai
2. Нажмите на профиль (правый верхний угол)
3. Settings → Skills
4. "Upload Skill"
5. Выберите .skill файл (v1.0 или v2.0 или обе)
6. Готово! ✅

### ШАГ 3: Использование

**Базовая v1.0:** Просто скажите "Создай checkpoint"  
**PRO v2.0:** Скажите "Авто checkpoint с валидацией"
