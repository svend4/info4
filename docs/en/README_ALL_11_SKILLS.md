## 📥 КАК УСТАНОВИТЬ (3 СПОСОБА)

### ✨ СПОСОБ 1: Автоматическая установка (ЛУЧШИЙ!)

**В Claude.ai:**

1. Открой Settings (⚙️ справа внизу)
2. Перейди в раздел "Skills"
3. Нажми "Upload Skill"
4. Выбери .skill файл (например, `chat-migration-refined-v36-compact.skill`)
5. Подтверди установку
6. ✅ Готово! Skill активирован!

**Время:** 30 секунд  
**Сложность:** Очень просто  
**Статус:** ⭐⭐⭐⭐⭐ РЕКОМЕНДУЕТСЯ

---

### 📱 СПОСОБ 2: Через Claude Desktop

**В Claude Desktop приложении:**

1. Settings → Skills
2. Add Custom Skill
3. Выбери папку или .skill файл
4. Подтверди
5. ✅ Готово!

**Время:** 1 минута  
**Сложность:** Просто  
**Статус:** ⭐⭐⭐⭐

---

### 💬 СПОСОБ 3: Через чат (если нет кнопки)

**В любом чате Claude:**

1. Распакуй .skill файл (получишь папку с SKILL.md)
2. Прикрепи SKILL.md к чату (кнопка 📎)
3. Напиши: "Install this skill"
4. Claude установит skill
5. ✅ Готово!

**Время:** 2 минуты  
**Сложность:** Средне  
**Статус:** ⭐⭐⭐
