# Archive / Оригинальная структура

Эта папка содержит документацию об оригинальной (плоской) структуре репозитория до реорганизации.

## Цель

Сохранить информацию о том, как выглядел репозиторий ДО реорганизации для:
- Сравнения структур
- Исторической справки
- Возможности отката (при необходимости)
- Понимания эволюции проекта

## Оригинальная структура

До реорганизации (2026-01-28) все файлы находились в корневой папке репозитория в плоской структуре:

```
info4/
├── *.skill                    # 88+ .skill файлов
├── *.md                       # 70+ markdown файлов
├── *.zip                      # 22 архива
├── checkpoint_creator.py      # 1 Python скрипт
└── README.md                  # Минимальный README
```

**Проблемы старой структуры:**
- Все файлы в одной папке (180+ файлов)
- Сложная навигация
- Смешение языков (EN/RU)
- Нет категоризации skills
- Архивы вперемешку с исходниками

## Новая структура

После реорганизации (2026-01-28):

```
info4/
├── skills/                    # Skills по уровням
│   ├── level-1-basic/
│   ├── level-2-intermediate/
│   ├── level-3-advanced/
│   ├── level-4-professional/
│   ├── special/
│   └── other/
├── docs/                      # Документация по языкам
│   ├── en/
│   ├── ru/
│   ├── checkpoints/
│   └── versions/
├── scripts/                   # Утилиты
├── releases/                  # Архивы
└── _archive/                  # Этот каталог
```

### Skills (101 файл)
**Было:** Все .skill файлы в корне
**Стало:** Организованы в 6 папок по уровням и категориям

### Документация (80+ файлов)
**Было:** Все .md файлы вместе, EN и RU смешаны
**Стало:**
- docs/en/ - английская документация

**Было:** В корне репозитория
**Стало:** releases/ с индексным файлом и описанием

### Скрипты (1+ файл)
**Было:** checkpoint_creator.py в корне
**Стало:** scripts/ со вспомогательными инструментами

## Сравнение структур

### До реорганизации
```bash
# Получить список всех файлов
ls -la

# Найти skill
ls *.skill | grep "smart-file"

# Найти документацию
ls *.md | grep -i "skill"
```

### После реорганизации
```bash
# Получить список skills по уровням
ls skills/level-1-basic/

# Найти skill
find skills/ -name "*smart-file*"

# Найти документацию
find docs/ -name "*skill*"
```

## Миграция

Если вы работали с оригинальной структурой:

1. **Старые пути все еще работают** - файлы не удалены
2. **Используйте новые пути** для лучшей организации
3. **См. индексные README** в каждой папке

### Примеры миграции путей

| Старый путь | Новый путь |
|-------------|-----------|
| `smart-file-organizer.skill` | `skills/level-1-basic/smart-file-organizer.skill` |
| `КАК_СОЗДАТЬ_СВОЙ_SKILL.md` | `docs/ru/КАК_СОЗДАТЬ_СВОЙ_SKILL_ПОЛНОЕ_РУКОВОДСТВО.md` |
| `MEGA_ARCHIVE_102_SKILLS.zip` | `releases/MEGA_ARCHIVE_102_SKILLS_COMPLETE.zip` |
| `checkpoint_creator.py` | `scripts/checkpoint_creator.py` |
