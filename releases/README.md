### Для новичков
Начните с **all-88-claude-skills.zip** - это сбалансированная коллекция основных skills.

### Для полноты
Скачайте **MEGA_ARCHIVE_102_SKILLS_COMPLETE.zip** для доступа ко всем skills и версиям.
