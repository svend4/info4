1. Перейдите на **https://claude.ai**
2. Войдите в аккаунт
3. Нажмите на **профиль** (правый верхний угол, ваша фотка/инициалы)

### ШАГ 3: Перейдите в Skills

1. В меню Settings найдите **Skills**

1. Нажмите **"Upload Skill"** (зелёная кнопка)

4. ✅ Появится сообщение "Skill uploaded successfully"
5. Вы увидите **"chat-migration-assistant"** в списке

### ШАГ 5: Установите v2.0 PRO

## 🆘 TROUBLESHOOTING

### Проблема: Skill не загружается

**Решение:**
1. Проверьте что файл .skill не повреждён
2. Убедитесь что расширение именно .skill
3. Попробуйте другой браузер
4. Очистите кэш браузера

### Проблема: Skills конфликтуют

**Не должно быть!** Все 3 версии совместимы.

Но если всё же:
1. Используйте explicit команды (см. "Как использовать")
2. Укажите версию явно: "Используй v2.0 PRO skill"
3. В крайнем случае удалите одну версию временно

### Проблема: Claude не использует skill

**Причины:**
1. Skill не активирован - проверьте в Settings → Skills
2. Команда не подходит - используйте trigger фразы выше
3. Claude не распознал контекст - будьте более явным

**Решение:**
Скажите явно: "Используй chat-migration-ultimate skill для создания checkpoint"
